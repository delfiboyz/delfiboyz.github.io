Red Team Operations: Malware Simulation Console

Welcome to the Offensive Cyber Simulation Lab. This environment is purpose-built for red team operators, malware researchers, and adversary emulation specialists to study advanced techniques, tactics, and procedures (TTPs).
All scenarios presented here are for authorized simulation, training, and educational use only.

💀 Simulated Malware Arsenal

  • RAT.exe – A full-featured remote access trojan, modeled after Quasar and njRAT.
  • PhantomStealer – Credential and data thief mimicking RedLine and Vidar.
  • WormInjector – A network-spreading payload leveraging SMBv1 EternalBlue vulnerabilities, similar to WannaCry.
  • CrimsonLoader – Polymorphic dropper using LOLBAS tactics and PowerShell stagers.
  • SleepyCat – Stealthy fileless implant using direct system calls and in-memory injection. Based on tools like ThreadStackSpoofer.

🎯 Simulated Attack Chain

  • Initial Access: Spear-phishing, malicious Office/HTA payloads from MalwareBazaar, USB HID attacks.
  • Execution: Living-off-the-land binaries (mshta, regsvr32, rundll32), obfuscated script execution.
  • Persistence: Scheduled tasks, registry run keys, WMI event consumers.
  • Privilege Escalation: Local CVEs, token impersonation, DLL sideloading. Source: Exploit-DB.
  • Defense Evasion: AMSI & ETW bypass, API unhooking, custom packers.
  • Exfiltration: Encrypted data channels over HTTPS, DNS tunneling, staging to simulated C2 endpoints.
