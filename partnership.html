Partnership

Kami telah bermitra dengan berbagai sekolah dan lembaga untuk memperkenalkan program coding kami kepada siswa.

Roadshow ke Sekolah Al Azhar

Kreafdemy telah melakukan roadshow ke berbagai ekstrakurikuler di Sekolah Al Azhar untuk memperkenalkan program coding kami kepada siswa. Dalam acara ini, siswa mendapatkan kesempatan untuk belajar langsung dari para ahli dan melihat berbagai proyek menarik yang dapat mereka buat dengan coding.

Roadshow ke Sekolah Surya Bangsa

Kreafdemy telah mengadakan roadshow ke Sekolah Surya Bangsa untuk memperkenalkan program coding interaktif kami. Para siswa sangat antusias mengikuti workshop yang kami adakan, di mana mereka belajar dasar-dasar pemrograman dan bagaimana menciptakan aplikasi sederhana. Roadshow ini bertujuan untuk menginspirasi siswa agar lebih tertarik dengan dunia teknologi dan coding.

Roadshow ke SDIT Al Huda

Kreafdemy telah melaksanakan roadshow ke SDIT Al Huda untuk mengenalkan program coding kami kepada para siswa. Di acara ini, siswa diajak untuk memahami konsep-konsep dasar coding melalui permainan edukatif dan proyek kreatif. Kegiatan ini bertujuan untuk menumbuhkan minat siswa terhadap teknologi sejak dini dan mendorong mereka untuk mengembangkan keterampilan coding.
